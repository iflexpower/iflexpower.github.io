IF-TP-6 is a simple wall-mount DMX controller designed for single- or multi-color control of low voltage LED products with a wired digital communication network. It takes a DC input from 24VDC and its data output is DMX.

Dimming tunable downlights made easy

IF-2CH-DRDL-L is a compact driver box, specially designed for driving all common COB LED downlights. The Low Current version (L) drives 10W to 30W within 12-48V.
IF-2CH-DRDL accepts 4 type of input signals, DMX-512A (RDM), DALI, 1-10VDC and manual dim which offers you compatibility in using it with various control systems.
The Low Current version can be set to operate in 350mA, 500mA and 900mA constant current output .

Dimming more tunable downlights
